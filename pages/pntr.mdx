<Callout emoji={<img src='https://github.com/RobLoach/pntr/raw/master/examples/resources/logo-55x55.png' className='h-10 w-10 !m-0' />}>[pntr](https://github.com/robloach/pntr) is a 2D image/drawing library for C. It's pronounced like "painter".</Callout>

With [pntr_app](https://github.com/robloach/pntr_app) you can target several graphics/sound/input libraries, and target most modern operating-systems, a libretro core, and the web. It's basically everything you need to make a 2D game that runs in most places. We use this extensively in [null0](null0).
